[h1]每日惊喜 (Daily Surprise)[/h1]

[h2]模组介绍[/h2]
每天日出时刻，系统会随机选择玩家获得一个特殊效果，可能是惊喜也可能是惊吓！为你的饥荒之旅增添随机性和乐趣！

[h2]主要特点[/h2]
[list]
[*] 每天随机选择玩家获得特殊效果
[*] 25种丰富多样的效果（15种惊喜 + 10种惊吓）
[*] 完全可配置的服务器信息过滤
[*] 支持多人游戏，自动同步效果
[*] 详细的系统提示
[/list]

[h2]惊喜效果示例[/h2]
[list]
[*] 超级速度：移动速度加倍
[*] 巨人化：体型变大
[*] 幸运日：额外掉落物
[*] 光合作用：站立回血
[*] 夜视能力：夜间视野提升
[*] 生物朋友：生物友好
[*] 彩虹光环：发光效果
[*] 星之祝福：星光护体
[*] 钓鱼达人：钓鱼增强
[*] 治愈光环：团队回血
[/list]

[h2]惊吓效果示例[/h2]
[list]
[*] 易碎玻璃：伤害加倍
[*] 饥饿加速：饥饿加快
[*] 感官失调：生命饥饿互换
[*] 方向混乱：移动反转
[*] 视野模糊：视野受限
[/list]

[h2]配置选项[/h2]
[list]
[*] BUFF持续时间（默认1天）
[*] 每日选择玩家数量（1-12人或全部）
[*] 惊吓效果开关及触发概率（默认30%）
[*] 调试信息显示等级
[/list]

[h2]注意事项[/h2]
[list]
[*] 效果每日自动刷新
[*] 玩家离线自动清除效果
[*] 服务器重启会重新应用效果
[*] 建议首次使用前备份存档
[/list]
